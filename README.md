# Student Grade Calculator v0.1

This program calculates final student grades based on homework and exam results.

## Final Grade Formulas

- **Average method**: Final = 0.4 × Homework Average + 0.6 × Exam  
- **Median method**: Final = 0.4 × Homework Median + 0.6 × Exam

## Features

- Manual input of students and grades
- Random generation of grades
- Reading data from a file (`Students.txt`)
- Sorting students by name
- Formatted table output with two decimal precision
- Handles any number of homework assignments

## Files

- `main.cpp` → The main program code  
- `Students.txt` → Example student data file
